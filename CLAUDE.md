# Regras do Projeto — Site Maria Eunice Santos

## Idioma
- Todo o conteúdo do site é em **Português Brasileiro**
- O código-fonte (CSS classes, comentários, nomes de variáveis, templates) deve ser escrito em **Português** sempre que possível
- Exceções: convenções do framework Jekyll (`_posts`, `_layouts`, `_includes`, `_config.yml`), atributos HTML padrão, propriedades CSS, e termos técnicos que até um brasileiro usaria em inglês (ex: "layout", "header", "footer", "grid", "flex")
- Nomes de arquivos de conteúdo devem ser em português com slugs (ex: `sobre-a-perversao.md`)

## Arquitetura
- Site estático usando **Jekyll** hospedado no **GitHub Pages**
- Sem banco de dados, sem backend, sem frameworks JavaScript
- CSS puro (sem Tailwind, Bootstrap, etc.) — um único arquivo `assets/css/estilo.css`
- JavaScript mínimo — apenas para busca no arquivo Vidas Abertas (`assets/js/busca.js`)

## Design
- Paleta: teal escuro `#2B5B5E`, dourado `#C9A96E`, fundo `#FAFAF7`, bege `#F0ECE6`, texto `#2D2D2D`
- Tipografia: Playfair Display (títulos), Source Sans Pro (corpo) — Google Fonts
- Mobile-first, responsivo
- Botão flutuante do WhatsApp em todas as páginas

## Estrutura de Coleções Jekyll
- `_publicacoes/` — artigos acadêmicos e ensaios (~20 textos)
- `_vidas_abertas/` — colunas do jornal Correio da Bahia (~276 textos)
- `_textos_franceses/` — textos acadêmicos em francês
